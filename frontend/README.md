# InnerLight V2 - Frontend

Mental Wellness Platform - React + Vite Frontend

## Quick Start

### Prerequisites
- Node.js 16+
- npm or yarn

### Installation

1. **Clone & Install**
```bash
cd frontend
npm install
```

2. **Configure Environment**
```bash
cp .env.example .env
# Edit .env with your API base URL
```

3. **Start Development**
```bash
npm run dev
```

Open http://localhost:3000

### Build for Production
```bash
npm run build
npm run preview
```
